Stable

There is no such thing as a stable release of RotiDeCode at this time.

Unstable

The canonical repository is hosted on Gitorious where you can easily follow the activity and browse the code

To clone the git repository:

{% highlight bash %} git clone git://gitorious.org/rotidecode/rotidecode.git {% endhighlight %}

You will need the following dependencies to build it:

  • Qt 4.6 or higher
  • QCodeEdit 3 (git MASTER)
  • CMake 2.6 or higher
